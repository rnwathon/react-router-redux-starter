## Clone this repo to start working

```
 git clone https://github.com/rnwathon/react-router-redux-starter.git
```

## Folder Structure

```
  src
    Components/
    Constant/
    Modules/
    Redux/
    Routes/
    index.js
    index.scss
```

## Components/
Components folder is a place for the collections of Presentational Components 

## Constant/
Constant folder is a place for the collection of constant variable that might be used everywhere

## Modules/
Modules folder is a place for the collections of Container Component or the pages of the actual web application

## Redux/
Redux folder is a place for redux configuration and custom middleware (if any)

## Routes/
Routes folder is a place for a list of routes or other custom route components (e.g PrivateRoute)

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).
